Fetching Results

Now that you have learned how to insert data into a database, its time to obtain some results from the database. Our query was an INSERT query for adding an employee into the database, however, since it was an insert query there is nothing to fetch from the database to display.

If you're interested in fetching the column names in a table you can do so with the statement handler following a prepared statement with a SELECT query:

$result = $stmt->fetchColumn();

If you're interested fetching the number of rows that were affected in a query, for example, lets say a DELETE query, you would use:

$count = $del->rowCount();

If you're interested in fetching all results, you could do:

$result = $stmt->fetchAll();

The function fetchAll() returns an array containing all the rows in the result set. This method is not recommended to be used to fetch large sets of results simce it uses high resources.

You may consider using fetch() instead to get individual results if you do not need all the rows in the result set. There are other functions to use to fetch results that are not mentioned here.
