You might noticed that in the connection code there are 4 parameters: mysql:host=$host, dbname=$dbname, $username, and $password.

For mysql:host=$host, you would need to replace $host with your database host that you are wanting to access and for dbname=$dbname, you would need to replace $dbname with your database name that you are wanting to access. This will be done in a separate file shown below.

For $username and $password, these two parameters represent your username and password to access your database. However, the $host, $dbname, $username and $password are passed in from a separate file. This is a way to protect your login information since it is not stored in this file.
