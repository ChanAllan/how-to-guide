Getting PDO on your system

PDO Requirements

If you already have PHP 5.1 or later installed on your system, you already have PDO. However, PDO can also be obtained in versions 5.0 and prior to 5.1 as a PECL (repository for PHP extensions)extension. PDO cannot be obtained in earlier versions prior to PHP 5.0 as PDO requires PHP 5.0's object oriented features.

Getting the necessary drivers

If you could not find pdo_mysql listed as enabled/installed from the phpinfo output, then you would need to install the PHP-MySQL driver.

This can be accomplished by running sudo apt-get install php5-mysql. If you run into any issues, you can refer to the official PHP documentation for troubleshooting the issue.

Now rerun the PHP file that contained "phpinfo( )" and determine if the driver was successfully installed. If done correctly, you can move on to the next step.
